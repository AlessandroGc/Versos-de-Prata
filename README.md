## Visão Geral

Loja virtual de joias em prata 925 com navegação completa por catálogo, carrinho de compras e finalização de pedido via WhatsApp.

## Funcionalidades

- Busca em tempo real e filtros por categoria
- Carrinho persistente com `localStorage`
- Finalização via WhatsApp com mensagem automática
- Design responsivo (mobile-first)
- Lazy loading e skeleton loading de imagens
- Cookie banner (LGPD)

## Stack

**HTML5 · CSS3 · JavaScript (vanilla) · localStorage**

## Estrutura
